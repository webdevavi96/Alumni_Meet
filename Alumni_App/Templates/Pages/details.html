Key Features

  • User Management:

    User Types: Supports three types of users: Alumni, Students, and Teachers.

    Authentication: Secure login and signup functionality with custom user models.

    Profile Management: Users can create and update their profiles, including personal details, social links, and profile pictures.

  • Blogs:

    Blog Creation: Alumni and teachers can share their experiences and insights by creating blogs.

    Blog Listing: A dedicated page to display all blogs with a preview of the content.

    Blog Details: Users can click on a blog to view its full content on a separate page.

  • Events:

    Event Management: Displays upcoming, ongoing, and past events.

    Event Details: Provides information about event dates, times, and descriptions.

  • Media Support:

    Users can upload profile pictures and blog images.

    Images are displayed dynamically across the platform.

  • Responsive Design:

    The application is fully responsive and works seamlessly across devices, including desktops, tablets, and mobile phones.

User Experience Flow

  • Home Page:

    The landing page introduces the platform and its purpose. Includes a navigation bar with links to Blogs, Events, and Profile pages.

  • Signup and Login:

    New users can sign up by providing their details, including user type (Alumni, Student, or Teacher). Existing users can log in securely using their email and password.

  • Profile Page:

    Users can view their profile details, including name, email, phone, address, and bio. Social media links (LinkedIn, GitHub, Instagram, etc.) are displayed if provided. Users can edit their profile or log out from this page.

  • Blogs:

    The Blogs page lists all blogs with a preview of the title, content, and author. Users can click "Read More" to view the full blog on the Details page.

  • Events:

    The Events page lists all events with their status (Upcoming, Ongoing, or Ended). Provides details about the event's title, description, date, and time.

Technical Details

  • Tech Stack:

    Backend: Django 5.2

    Frontend: HTML5, CSS3, Bootstrap 5

    Database: SQLite

    Deployment: WSGI/ASGI

  • Project Structure:

    Alumni_App: Contains models, views, templates, and static files.

    Alumni_Meet: Contains project-level settings, URLs, and configurations.

    Media: Stores uploaded images (e.g., profile pictures, blog images).

    Static: Contains CSS, JavaScript, and other static assets.

Future Enhancements

  • Messaging System: Enable users to send messages to each other.
  • Event Registration: Allow users to register for events directly from the platform.
  • Notifications: Notify users about new blogs, events, or updates.
  • Admin Dashboard: Provide an admin interface for managing users, blogs, and events.
